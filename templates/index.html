Tutorial

Select Input Device

Select your sound source from:

  • Keyboard - An oscillator controlled by a virtual keyboard interfaced with the mouse or computer keyboard
  • Microphone - External audio stream from Microphone or other audio device

Once chosen configure your input device by selecting the oscillator waveform or audio sample rate

Oscillator Waveform

Waveforms named after their shape are as follows:

  • Sine - Smooth and pure, resembling a fundamental tone
  • Square - Pronounced harmonics, generating a bright, buzzy tone.
  • Sawtooth - Rich in harmonics, producing a bright and edgy sound.
  • Triangle - Mildly harmonic, with a soft, mellow timbre.

Sample Rate

Select the sample rate used by your audio device.

Set Gain

The Gain sets the maximum volume reachable by the input device

ADSR Envelope

If using the oscillator the adsr controls how the volume changes overtime with each note press.

  • Attack: Determines how quickly the sound reaches its peak volume.
  • Decay: Sets the time it takes for the sound to decrease from the peak to the sustain level.
  • Sustain: Defines the level at which the sound remains as long as the note is held.
  • Release: Controls the time it takes for the sound to fade out after the note is released.

Filters

Add Filters to modify the sound. Each filter has its own parameters to set which change how the filter functions
